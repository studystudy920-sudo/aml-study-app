# GENSPARK スライド作成指示書（修正版）

## 目的
犯罪収益移転防止法施行規則の改正（2027年4月1日施行）における、オンライン本人確認（eKYC）方式の選択フローが「変更前」と「変更後」でどう変わるかを、判定フローチャート形式で2枚のスライドにまとめる。

---

## スライド1：変更前の判定フロー（現行制度 〜2027年3月31日）

### タイトル
「現行：オンライン本人確認（eKYC）の判定フロー」

### フローチャート構成（上から下へ流れる）

```
[START] 非対面で本人確認が必要
         │
         ▼
    ┌──────────────────┐
    │ 本人確認書類は    │
    │ ICチップ付きか？  │
    └──────────────────┘
      │YES          │NO
      ▼              ▼
 ┌─────────┐    ┌──────────────────┐
 │ICチップを│    │ 顔写真付き       │
 │読取可能か│    │ 書類があるか？   │
 └─────────┘    └──────────────────┘
  │YES    │NO     │YES         │NO
  ▼       ▼       ▼             ▼
┌────┐ ┌────┐  ┌────┐      ┌─────────┐
│ヘ方式│ │ワ方式│ │ホ方式│     │書類の写し│
│    │ │(JPKI)│ │★主流│     │を郵送    │
└────┘ └────┘  └────┘      └─────────┘
ICチップ  電子    書類撮影     郵送
+容貌    証明書   +セルフィー  （時間大）

         └───┬───┘
             ▼
        ┌─────────┐
        │ ト方式   │
        │（代替）  │
        └─────────┘
        他の特定事業者
        への照会
```

### 補足テキスト
- ★ ホ方式が市場の主流（約7〜8割のシェア）
- 選択肢が多く、事業者ごとに採用方式がバラバラ
- 書類の写し郵送は数日〜1週間かかる

### デザイン指示
- フローチャートの各判定ノード：菱形（ダイヤモンド形）
- 各方式のゴールノード：角丸四角形
- ホ方式のゴールノード：青色で強調、★マーク付き「主流」
- ワ方式・ヘ方式：通常の青色
- 書類郵送：グレーで「レガシー」感を出す
- 全体の流れ：上から下、分岐は左右

## スライド2：変更後の判定フロー（2027年4月1日〜）

### タイトル
「改正後：オンライン本人確認（eKYC）の判定フロー — 2027年4月1日施行」

### フローチャート構成（上から下へ、シンプルに）

```
[START] 非対面で本人確認が必要
         │
         ▼
    ┌──────────────────────┐
    │ マイナンバーカードを  │
    │ 持っているか？        │
    └──────────────────────┘
      │YES              │NO
      ▼                  ▼
 ┌──────────────┐    ┌──────────────────┐
 │ JPKI読取     │    │ ICチップ付き     │
 │ 可能か？     │    │ 本人確認書類は   │
 │（スマホ等）  │    │ あるか？         │
 └──────────────┘    └──────────────────┘
  │YES       │NO      │YES         │NO
  ▼           ▼        ▼             ▼
┌──────┐  ┌──────┐  ┌────┐     ┌─────────┐
│カ方式 │  │ヘ方式│  │ヘ方式│    │ ト方式   │
│(旧ワ) │  │      │  │      │    │          │
│◎原則 │  │      │  │      │    │          │
└──────┘  └──────┘  └────┘     └─────────┘
JPKI      ICチップ   ICチップ     他の特定
電子証明書 +容貌     +容貌       事業者照会
【最も推奨】

    ✕ ホ方式（書類撮影+セルフィー）→ 廃止
    ✕ 書類の写し郵送 → 廃止
```

### 変更前との違いを強調するポイント

1. **フローがシンプルになった**：選択肢が減り、迷いにくい
2. **入口がマイナンバーカードの有無**：最初の判定が変わった
3. **カ方式（JPKI）が最優先・原則**：フローの最も左（推奨パス）に配置
4. **ホ方式・郵送が消えた**：フロー図上に「✕ 廃止」として薄くゴースト表示

### 補足テキスト（スライド下部）
- 公布日：2025年6月24日 ／ 施行日：**2027年4月1日**
- 背景：偽造書類・ディープフェイクの脅威、FATF第4次対日相互審査
- カ方式（JPKI）は偽造・なりすましが原理的に不可能

### デザイン指示
- カ方式のゴールノード：緑色で大きく強調、◎マーク付き「原則方式」
- ヘ方式：通常の緑色
- ト方式：薄い緑色（補助的位置づけ）
- 廃止された方式：赤の取消線＋薄いグレーでゴースト表示（フロー外に配置）
- 全体のフローが変更前より**短く・シンプル**であることが視覚的に伝わるように
- 変更前スライドと並べたときに「複雑 → シンプル」の対比が明確になるように

---

## 全体の共通指示

- **フォーマット**: 横向き（16:9）スライド 2枚
- **形式**: フローチャート（判定フロー図）
- **フォント**: ゴシック体（Noto Sans JP等）、タイトル24pt以上、ノード内テキスト12〜14pt
- **色使い**:
  - 判定ノード（菱形）：濃い青の枠線
  - 変更前ゴール：青系
  - 変更後ゴール：緑系（カ方式は濃い緑で強調）
  - 廃止：赤系＋グレー
- **矢印**: YES/NO のラベル付き、流れが明確に
- **アイコン**: フローの各ゴールノードに小さなアイコン（スマホ、ICチップ、マイナンバーカード、銀行ビル等）
- **言語**: 日本語
- **出典表記**: スライド右下に「出典：犯罪収益移転防止法施行規則（令和9年4月1日施行改正）」

## 最も重要なポイント
**2枚を並べたとき、変更前は「分岐が多く複雑」、変更後は「JPKI中心でシンプル」という対比が一目で伝わること。**
